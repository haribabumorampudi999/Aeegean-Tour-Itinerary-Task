**Aegean Tour Itinerary Task**

**Prerequisites**

Install the latest version of Python (Python 3.x recommended).

Ensure Python is added to your system PATH.

**Setup Instructions**

Clone the repository or download the source code.

https://github.com/haribabumorampudi999/Aeegean-Tour-Itinerary-Task.git

Navigate to the project directory.

Open a Command Prompt or Terminal from the source code location.

**Running the Application**

**Option 1:** Run with Input File
python .\Aeegean_Tour_Itinerary_task.py .\input.txt

**Option 2:** Run with Manual Input
python .\Aeegean_Tour_Itinerary_task.py

After running the command:
Enter the customer data manually.
Press Enter.
Press Ctrl + Z (Windows) to signal end of input.
Press Enter again to execute.

Notes
Make sure input.txt exists in the same directory when using Option 1.
For non-Windows systems, replace Ctrl + Z with Ctrl + D.
